Praxis Favicon Generator

Click the buttons below to generate and download favicon files.

Instructions:

  1. Click "Generate All Favicons" to create all required favicon sizes
  2. Right-click each canvas and "Save image as..." to download
  3. Save files to the public/ directory with these names:
    • favicon-16x16.png
    • favicon-32x32.png
    • apple-touch-icon.png (180x180)
    • android-chrome-192x192.png
    • android-chrome-512x512.png
    • og-image.jpg (1200x630)
  4. Also create favicon.ico from the 32x32 PNG using an online converter
